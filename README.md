# 📒 Contact Management System (C Program)

A simple **Contact Management System** written in C that allows users to:

- Add new contacts (Name + Phone Number)
- Search for contacts by phone number
- Delete contacts by phone number
- Display all saved contacts

This project demonstrates the use of arrays, strings, and basic file handling concepts in C.

---

## 🚀 Features
- Add multiple contacts in one go
- Search contacts quickly by phone number
- Delete contacts with proper shifting of array elements
- Display all contacts in a neat list
- Menu-driven interface for easy navigation

---

## 🛠️ Technologies Used
- **Language:** C
- **Compiler:** GCC / Any standard C compiler
- **Concepts:** Arrays, Strings, Loops, Functions

## ▶️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/contact-management.git
   ```
2. Navigate to the project folder:
   ```bash
   cd contact-management
   ```
3. Compile the program:
   ```bash
   gcc main.c -o contact_manager
   ```
4. Run the executable:
   ```bash
   ./contact_manager
   ```
